### default (기본) 연산자

default(형식 Type, 형식 매개 변수 T  )  

```cs
default(int) // 0
default(object) // null
default(int?) // null
default(System.Numerics.Complex) // (0, 0)
default(System.Collections.Generic.List<int>) // null
```

### default (기본) 리터럴
